# TextArea-Message
The project creates a character counter that functions in real time for textarea elements using HTML and CSS and JavaScript. Users can type messages while the system displays the number of characters used and the remaining limit. The counter updates instantly and shows a warning when the maximum character limit is reached.

# Features

Real-time character counting while typing

Displays characters used and characters remaining

Shows a warning message when the limit is reached

Uses the input event listener for instant updates

Simple and clean user interface

# Technologies Used

HTML5 – Structure of the page

CSS3 – Styling and layout

JavaScript (DOM Manipulation) – Real-time counter functionality

# Purpose

The main goal of this project is to practice textarea message count, DOM manipulation, and user input handling in JavaScript.
